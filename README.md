# GitHub Weekly🤖


## 介绍
经常关注ruanyf分享的科技内容,推荐或自荐文章/软件/资源实时更新在GitHub的issue
阅读起来不是很友好，于是尝试让Cursor写一个自动更新的站点。

## 实现
- 定时抓取数据到json文件
- Vue构建静态站点部署到GitHub
- GitHub Action定时更新同步
- 支持滚动加载和分页模式
